## 📌 Description

This is a simple **Random Joke Generator Web Application** that fetches and displays a random joke when the user clicks a button.
It uses an external API to provide fresh jokes every time.

## 🚀 Features

* Generate random jokes instantly
* Uses real-time API
* Simple and user-friendly interface
* Error handling for failed requests

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Joke API (https://v2.jokeapi.dev/)

## ▶️ How to Run

1. Clone the repository
2. Open the project folder
3. Run the file:
   index.html
